# LiteTrack

轻量级网站访问统计，可在自己项目中自由使用。

## 这是什么

LiteTrack 提供简单的网站访问统计能力：接入一个小脚本，即可在后台查看页面的访问量、每日趋势和热门页面，无需复杂配置。

适合个人站、博客、文档站等需要了解「谁在访问、看了哪些页面」的场景。

## 在线使用

- **产品地址**：<https://litetrack.xiaowo.live>
- **SDK 文件**：可在 [apps/sdk/dist/](./apps/sdk/dist/) 查看 SDK 文件，放入你的项目并通过 script 标签引入即可使用。

使用方式与接入说明见 [SDK 文档](./apps/sdk/README.md)。
